## Resumen
Se desarrolla **PodWave Mobile Lab**, una app móvil web propia basada en el patrón de clase (clon Spotify con pantallas) y ampliada con framework de navegación animada, API REST y persistencia completa.

## 1) Modificaciones estéticas y visuales
- Diseño mobile-first para viewport de smartphone.
- Navegación inferior fija estilo aplicación nativa.
- Transiciones animadas entre pantallas con framework JS propio.
- Tarjetas y tablas compactas orientadas a interacción táctil.

## 2) Modificaciones funcionales (alto calado)
- Backend Flask + SQLite relacional.
- Registro de usuario y sesiones persistentes.
- Catálogo dinámico de canales y episodios con filtros.
- Favoritos persistentes por usuario.
- Telemetría de eventos de navegación y reproducción.
- Métricas globales y ranking de uso.

## 3) Evidencia de segundo curso
La actividad incorpora arquitectura completa cliente-servidor, modelado de datos y analítica de interacción, superando ampliamente una maqueta frontend.
